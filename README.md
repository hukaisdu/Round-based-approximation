# Structure 

This filefolder contains source codes for Ascon, Present, Koala and Subterranean. 

We recommand readers read Present first, because it is the simpliest and most straightforward. 

For Ascon, the codes are c++, so you should use,  for example, 

'''
g++ ascon1st.cpp -o ascon1st -std=c++11 -O3 
'''
to compile them. 

Some codes use multi-threading for speeding up, please change the number of threads according to your system to achieve a better speed.
